Langkah 1 — Buka Apps Script
Buka Google Sheets Anda, klik menu Extensions → Apps Script. Hapus semua kode yang ada, lalu tempel kode berikut:
Langkah 2 — Deploy sebagai Web App
Klik Deploy → New deployment → pilih type Web app. Atur:
• Execute as: Me
• Who has access: Anyone
Klik Deploy, izinkan akses jika diminta, lalu salin URL deployment-nya.
